Question 1: What is Dart? (Dart là gì?)
    Dart is a programming language that is developed by Google. It is a client-optimized language for fast apps on any platform. Dart is an object-oriented, class-based, garbage-collected language with C-style syntax. Dart can compile to either native code or JavaScript.
    (Dart là ngôn ngữ lập trình do Google phát triển. Nó là một ngôn ngữ tối ưu cho khách hàng để tạo ra các ứng dụng nhanh trên bất kỳ nền tảng nào. Dart là một ngôn ngữ có hướng đối tượng, dựa trên lớp, thu thập rác với cú pháp C. Dart có thể biên dịch thành mã máy hoặc JavaScript.)
